# Android EVPN
Android App VPN Proxy - EVPN

The evpn is an android security application which allow users to keep their privacy safe by using secured proxy and openvpn protocol

Features Include

* VPN Tunnel Service
* Set Timer
* Speed Test
* Stop Watch
* Boost Alarm
